# Demo package

`demo-master.mp4` is a 1920×1080, H.264, fast-start, caption-burned recording of public guest flows. `demo-captions.srt` and `transcript.md` match its 52-second cut. The accompanying YouTube thumbnail is derived from the truthful public homepage capture.

Regenerate only from `https://arcticrss.com` with `PH_LAUNCH_GATE=GO` and `PH_ASSET_CAPTURE_AUTHORIZATION=PUBLIC_AND_DEMO_DATA_ONLY`; never record an account, private feeds, or credentials.
