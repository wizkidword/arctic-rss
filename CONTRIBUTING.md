# Contributing

Thanks for helping build Arctic RSS.

## Development

- Keep feed fetching secure by default.
- Scope all user-owned data by `userId`.
- Add tests before helper or domain behavior changes.
- Run `npm run test`, `npm run typecheck`, and `npm run lint` before opening a PR.

## Scope

The project is milestone-driven. Avoid adding feed parsing, AI behavior, or billing inside foundation work unless the milestone explicitly calls for it.
